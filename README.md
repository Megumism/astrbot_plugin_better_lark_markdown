# astrbot-better-lark-markdown

这是一个面向飞书的增强版 Markdown 解析器，支持更多的 Markdown 语法和功能。

> 注意，本插件不作用于“流式输出”，因为在使用 CardKit 流式卡片时，大部分Markdown语法都能得到正确解析。但如果模型输出有图片链接，很大概率导致流式更新文本失败，吞没后面所有消息。
> 是否启用“分段发送”消息对插件部分设置也有影响。

## 目前支持的 Markdown 语法

- 所有飞书原生支持的 Markdown 语法（详见下文）
- 表格，转换为飞书卡片发送
    - 卡片原生支持表格对齐

以下是支持目标

- [ ] 下划线
- [ ] 任务列表
- [ ] 图片

## 附：飞书原生可处理的 Markdown 语法列表

- [x] 各级标题
- [ ] 水平分割线（支持`---`、`***`，不支持`* * *`）
- [ ] 段落文本
- [x] 粗体
- [x] 斜体
- [x] 删除线
- [x] 粗体斜体
- [x] 行内代码
- [x] 代码块
- [x] 指定代码语言的代码块
- [x] 无序列表
- [x] 有序列表
- [x] 嵌套列表
- [x] 引用块
- [x] 嵌套引用
- [x] 超链接
- [ ] 自动链接（自动转译链接，但不会忽略`<>`）
- [ ] 转义字符（不会忽略`\`）
- [ ] 删除线嵌套样式
- [ ] 行内公式（视平台支持）
- [ ] 块级公式（视平台支持）
- [x] Emoji 表情
- [ ] 特殊符号
- [ ] 锚点链接（视平台支持）
- [ ] 多段落内容
- [ ] 混合语法排版

> **不考虑支持的**
>
> - [ ] 注脚
> - [ ] HTML 标签嵌入
> - [ ] 折叠内容（details / summary）
> - [ ] 非数学环境上下标（1~2~ 2^10^）
